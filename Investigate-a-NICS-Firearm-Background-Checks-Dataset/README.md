# DAND Projects (Udacity Course)
# Investigate a NICS Firearm Background Checks Dataset

## Project Overview
This project analyzes data from the FBI's National Instant Criminal Background Check System (NICS). The goal was to understand how firearm demand fluctuates over time and across different U.S. states.

## Key Insights
- **Growth Trends:** Total background checks have increased significantly over the last two decades.
- **2020 Spike:** There was a record-breaking surge in background checks in 2020.
- **State Variability:** Different states show unique patterns based on local permit laws and recheck requirements.

## Data Limitations
- **Not a Sales Tracker:** A background check does not guarantee a final sale occurred.
- **Private Sales:** The dataset does not include private person-to-person transfers in many states.
- **Process Differences:** States like Kentucky have high numbers because they perform monthly rechecks on permit holders.
